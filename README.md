# Tez Ter

**Tez Ter** — foydalanuvchining klaviaturada yozish tezligini tekshiruvchi interaktiv veb ilova. Loyiha yordamida foydalanuvchi ma’lum vaqt ichida berilgan matnni teradi va yakunda yozish tezligi, aniqlik darajasi hamda xatolar soni hisoblanadi.

## Loyiha maqsadi

Ushbu loyihaning asosiy maqsadi foydalanuvchining yozish tezligini o‘lchash, xatolarni real vaqt rejimida ko‘rsatish va natijalarni saqlab borishdan iborat.

Loyiha HTML, CSS va JavaScript texnologiyalari yordamida ishlab chiqilgan.

## Asosiy imkoniyatlar

- 30, 60 va 120 soniyalik test rejimlari
- Oson, o‘rta va qiyin matn darajalarini tanlash
- Foydalanuvchi ismini kiritish
- Real vaqt rejimida yozilgan belgilarni tekshirish
- To‘g‘ri va noto‘g‘ri harflarni rang bilan ajratib ko‘rsatish
- WPM — bir daqiqadagi so‘zlar sonini hisoblash
- CPM — bir daqiqadagi belgilar sonini hisoblash
- Aniqlik foizini hisoblash
- Xatolar sonini ko‘rsatish
- Progress bar orqali test jarayonini kuzatish
- Qorong‘i va yorug‘ rejim
- Yakuniy natijani nusxalash
- Eng yaxshi 5 ta natijani saqlash
- Natijalarni `localStorage` orqali brauzerda saqlash
- Saqlangan natijalarni tozalash
- Mobil qurilmalarga mos responsive dizayn
